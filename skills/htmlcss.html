PERSONAL WEBSITE

This is my first attempt at HTML and CSS. Prior to this website, I had zero experience! HTML, CSS and JavaScript are all a large work in progress for me; however I have been thoroughly enjoying learning another new skill. Especially one I believe will be valuable to me in the future.

Some skills I learned while building this website:

  • Plan and build CSS webpage layouts
  • Style pages and text using CSS and HTML
  • Understand box models padding and positioning
  • Create and style forms for gathering user information
  • Position images and website links

Luckily, I had some basic knowledge of C, C++ and Arduino programming prior to starting this project. These skills helped me understand the code I was reading in books, and in online tutorials.

I started from the ground up with a simple background image and some overlaid text. From there, I added a fixed top navigation bar which is responsive based on what page is selected. A website footer was finished next. From here, I began building individual pages from the visions I had on paper. I knew I wanted my name to stand out on my home page, but regular text wasn’t going to cut it.

I used free graphic design software called InkScape to create a vector image. This allowed me to make a white background with see through text for my home page. I previously gained experience with InkScape when I was creating G-Code for my Arduino CNC project.

I also learned how to use images to link to various pages such as Instagram, and email. Furthermore, these links can also direct a user to child pages on my website.

For my “skills” page, I learned how to create tables and size images to suit. The major challenge with this was formatting the page for various window sizes. It was challenging to create responsive images that resize themselves based on the size of the device used.

By this point I didn’t really even have any “content” on my website and I was really thinking that creating my own website on WordPress would be much less of a pain. However, solving problems absorbs me and I really enjoy when the lightbulb clicks and something I have been struggling with suddenly works. Success! Sure enough, solving small problem after small problem creates something complex and attractive.

I learned how to create a form for collecting visitor information. The problem with this is, that for the form to work correctly, it needs to have some PHP code which will send an email directly from the form. I don’t know the first thing about PHP!

Something I learned quickly is that planning is possibly the most important part of building a website. With a solid plan, building a vision is infinitely easier.

Out of all of the personal projects I have undertaken, I would argue that this has website been the most rewarding so far.
